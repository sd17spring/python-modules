# python-modules

Practice with organizing larger Python programs as modules

Check out the [course website](https://sd17spring.github.io/day/day-21/) for more details.

## Installation

Install pygame:

    pip3 install -r requirements.txt

## Usage

    python3 brick_breaker.py
